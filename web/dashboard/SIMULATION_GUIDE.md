# Dashboard Simulation Guide

The dashboard is currently running in **Simulation Mode**. No database connection is required.

## How to Login
1. Navigate to the Login Page (or root ).
2. Enter **any** valid email format (e.g., `admin@test.com`).
3. Enter **any** password.
4. Click **Sign In**.

The system simulates a successful login and grants you a **Free Tier** session by default.

## How to Test Subscription Features
1. Once logged in, go to the **Settings** page.
2. Select **Pro Plan** or **Enterprise Plan**.
3. The dashboard will instantly update to unlock premium features:
   - **Analytics**: Unlocked
   - **Workflows**: Unlocked
   - **Documents**: Unlimited uploads

## Troubleshooting
If you are redirected back to login:
- Ensure your browser accepts local storage.
- Try refreshing the page.
